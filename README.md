# Open AI Translator

The main purpose is to allow a simple application to have open AI completing common questions and answers.

## How to Run

- Step 1: Make sure you have Python

- Step 2: Install the requirements: `pip install -r requirements.txt`

- Step 3: Go to this app's directory and run `python app.py`



## Details about This App

There are three tabs in this app.

- **Public**: this is a page which can be accessed by anyone, no matter if the user has logged in or not.

- **Private**: Only logged-in user can access this page. Otherwise the user will get a 401 error page.

- **Admin Page**: This part is private to /admin , which only allow for admin users
